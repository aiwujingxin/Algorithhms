归并排序就是先把左半边数组排好序，再把右半边数组排好序，然后把两半数组合并.

要对子数组 a[1o..hi] 进行排序，先将它分为 a[1o..mid] 和 a[mid+1..hi] 两部分，分别通过
递归调用将它们单独排序，最后将有序的子数组归并为最终的排序结果。
